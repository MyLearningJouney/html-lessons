Lets Talk About HTML

What is HTML ?

HTML stands for Hypertext Markup Language

It's the standart markup language to create web pages.

Consists in a series of elements and tags that tells to the browser how to display the content.

Works in a tree shape.

Tags from HTML:

Each HTML Tag represents semantics, or meaning in the web page structure. Bacause of them, the brosers know how to display content correctly.

Let's see some tags below:

  • <!DOCTYPE>: This declaration informs to the browser the document type.

    Usage:

    <!DOCTYPE html>

  • <html>: this tag represent the root of an HTML Document, here we can also tell to the browser the language of the web page.

    Usage:

    <html lang="pt-br">
